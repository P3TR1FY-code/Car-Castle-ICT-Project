#  Car Castle – Car Showroom Website

##  Project Description
Car Castle is a frontend-based car showroom website developed as a semester project for  
**Applications of ICT Lab (CS181L)** at **Air University Islamabad**.

The project is designed using **HTML, CSS, and JavaScript only**, without using any templates,
CMS, or paid libraries, strictly following the course guidelines.

##  How to Run the Project
1. Download or clone the repository.
2. Extract the folder (if zipped).
3. Open the project folder.
4. Double-click **index.html**.
5. The website will open in a web browser.
